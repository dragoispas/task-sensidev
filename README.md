# Rick and Morty App

Features a search bar with a dropdown (that filters for status -> alive / dead / unknown) that does server-side filtering on characters.

Any character card can be clicked to view more details about them. 

**Built using:**
- React
- Typescript
- Redux
- React Router
- Styled Components
